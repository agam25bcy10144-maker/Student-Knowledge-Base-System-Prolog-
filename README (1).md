# Student-Knowledge-Base-System-Prolog-
<br>.Overview

<br>This project is a simple Student Knowledge Base System written in Prolog.
<br>It stores student information (name, age, and course) and provides features to:
<br>Retrieve student information
<br>Search by course
<br>Add or remove students
<br>Generate simple reports
<br>This project is suitable for beginners learning Prolog and knowledge-base systems.
<br>Features
<br>1. Data Retrieval
<br>Get a student's age and course by their name.
<br>Find all students enrolled in a specific course.
<br>2. Data Update (Dynamic Facts)
<br>Add new students to the database.
<br>Remove existing students.
<br>3. Reporting & Analytics
<br>Count the total number of students.
<br>List all students with their details.
